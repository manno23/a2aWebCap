- **Yellow text** = Working state, warnings, in-progress
- **Cyan text** = Task creation, API calls, submitted state
- **Magenta text** = Special states (input-required, auth-required)
- **Gray text** = Metadata, timestamps, secondary info
- **Purple text** = Artifacts, data structures
- **Orange text** = Auth-required state
- **Blue text** = Database operations, system events
- **Bold white** = Assertion expectations
